# ThicMobKai Converter

Trang web thử nghiệm để quét pack MythicMobs + ModelEngine 4 và sinh cấu hình ThicMobKai.

## Hiện có

- Kéo thả pack `.zip`
- Quét file mob / boss / model
- Xuất ra bản nháp `mobs.yml` và `bosses.yml`
- Gợi ý lỗi thường gặp

## Bản sau

- Hỗ trợ `.rar`
- Sửa YAML lỗi sâu hơn
- Xuất zip cấu hình hoàn chỉnh
- Tách riêng wizard cho ModelEngine 4 / MythicMobs
