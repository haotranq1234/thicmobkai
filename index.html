Toàn bộ dungeon, mob, boss, danh hiệu, và bảng điều khiển đều ở cùng một nơi.

Đây là trang hướng dẫn chính thức cho plugin ThicMobKai. Mục tiêu của web là giúp người chơi hiểu nhanh, quản trị viên chỉnh nhanh, và người mua plugin nhìn ra được toàn bộ giá trị của hệ thống mà không phải mò từng file YAML.

Xem từng trang Xem file cấu hình
JAR ThicMobKai-1.0.0.jar Phiên bản Việt hoá, không còn nhánh zkai.
Trọng tâm Mob - Boss - Dungeon Bãi farm, boss thế giới, rương boss, danh hiệu, skill, card.
Hỗ trợ ModelEngine 4 · DiscordSRV · PAPI ServerBoardKai hook, model pack, tab/chat placeholder.

Trang này có gì

  • Tổng quan nhanh về plugin và mục tiêu thiết kế.
  • Danh sách hệ thống gameplay: mob, dungeon, boss, world boss, chest.
  • Giải thích file YAML để admin chỉnh không cần build lại.
  • Tích hợp với ModelEngine 4, DiscordSRV, ServerBoardKai và PlaceholderAPI.

Phong cách triển khai

  • Tối ưu cho server RPG / dungeon / farm mob.
  • Ưu tiên GUI rõ ràng, lore ngắn, đẹp và dễ quản trị.
  • Chia tính năng theo từng trang để người mới không bị ngợp.
  • File config tách riêng để tự chỉnh ngay trong host.

Điểm nổi bật

  • Boss thế giới rơi rương top 1-5.
  • Dungeon hub riêng, card riêng, bảng top riêng.
  • Danh hiệu cho chat và tablist qua placeholder.
  • Holo spawner và board bên phải có thể hook theo vùng.

Trang 2

Hệ thống gameplay

Đây là phần dành cho người chơi và admin muốn hiểu plugin đang làm gì. Mỗi khối hệ thống đều có mục đích riêng, tránh nhồi tất cả vào một menu.

Dungeon Hub

Quản lý lượt, card, bảng top và vùng dungeon

Người chơi mở dungeon hub để xem danh sách dungeon, lượt còn lại, mua thêm lượt, và đi thẳng vào khu phù hợp.

/tmk dungeon
/tmk dungeon top
/tmk dungeon buy <id> [amount] [money|playerpoints|xucorekai]
Mob Farm

20 mob bãi farm riêng, 100% item trao đổi

Mỗi mob farm có level, mob vanilla hợp lý, và vật phẩm rơi để đổi đồ tại /warp tradedungeon.

mobs:
  desert_sentinel:
    entity-type: HUSK
    level: {min: 1, max: 20}
    drops:
      - item-id: sand_token
Boss Book

Boss thường, boss ẩn và boss thế giới tách riêng

Sổ boss cho người chơi xem tiến trình; boss thế giới dùng hệ thống top dame và rương riêng.

world-bosses.yml
bosses.yml
hidden-bosses.yml
world-boss-chests.yml
World Boss Chest

Boss thế giới rơi rương cho top 1-5

Boss chết sẽ đặt rương ở chỗ hạ boss. Top 1 đến top 5 mở rương để nhận quà riêng theo rank.

Top 1 → quà mạnh nhất
Top 2-5 → pool riêng
Hologram → báo tên boss và số lượt đã mở
Danh hiệu

Hiện trong chat, tablist và lore hồ sơ

Danh hiệu có thể cấp bằng quyền admin hoặc mở bằng thành tích. Dùng placeholder để ghép với plugin khác.

%rank% %player% %title%
chat enabled: false/true
tab enabled: false/true

Trang 3

Cấu hình & file YAML

Mục này dành cho admin. Tất cả tiêu đề, lore, mô tả lệnh, và cách hiển thị GUI đều có thể chỉnh trong file host mà không cần build lại plugin.

Danh sách file chính

  • config.yml - cấu hình tổng, top dame, world boss, bảng màu, hook.
  • message.yml - đổi tên GUI, lore, mô tả lệnh, câu hướng dẫn.
  • farm-mobs.yml - mob bãi farm 1-20.
  • dungeon-mobs.yml - mob riêng cho dungeon.
  • bosses.yml - boss thường.
  • world-bosses.yml - boss thế giới.
  • world-boss-chests.yml - pool rương top 1-5.
  • hidden-bosses.yml - boss tinh anh / boss ẩn.
  • dungeons.yml - rank, thưởng, vùng dungeon.
  • spawners.yml - khu spawner và holo.
  • card-rewards.yml - pool quay thưởng từ card.
  • titles.yml - danh hiệu.
  • collections.yml - bộ sưu tập dungeon.
  • secret-treasures.yml - kho báu bí mật.

message.yml dùng để làm gì

Đây là file để admin tự đổi phần chữ mà người chơi nhìn thấy, bao gồm:

Tiêu đề GUI Lore item Mô tả lệnh Hướng dẫn slot Tên khu vực Dòng mô tả boss 
text-overrides:
  gui-title:
    dungeon-hub:
      vi: "&8ThicMobKai &7- &bTrung tâm hầm ngục"
  help:
    member:
      vi:
        - "&f/tmk dungeon"
        - "&7Mở trung tâm hầm ngục."

Lưu ý cho admin

  • Không hardcode lore trong code nếu đã có mục override trong message.yml.
  • GUI con nên chỉ giải thích tính năng của GUI đó, không nhét hướng dẫn trùng ở slot 53.
  • Bộ boss thế giới hiện có top dame, boss bar, DiscordSRV, và rương riêng.
  • Những file item / drop có thể chỉnh bằng kéo-thả item vào GUI quản trị.

Trang 4

Tích hợp plugin

Đây là phần để ghép ThicMobKai vào server hiện có. Trang này nói rõ plugin nào hỗ trợ gì, cái nào nên bật, và phần nào chỉ là hook để hiển thị đẹp hơn.

ModelEngine 4

Giữ nguyên model `.bbmodel`, gắn id model vào mob, và dùng hiệu ứng của pack để tạo chuyển động / vung kiếm / đòn đánh.

DiscordSRV

Boss thế giới và boss ẩn có thể báo riêng, kênh riêng, tránh gộp chung với thông báo mob farm.

PlaceholderAPI

Dùng cho board, tablist, hologram, danh hiệu, cấp người chơi, combo, lucky score và top dame.

ServerBoardKai

Board bên phải đổi theo vùng: dungeon, boss, world boss, farm, spawn, trade. Có thể tự sinh config cho từng khu.

MythicMobs pack converter

Web dùng để đọc pack bên ngoài, giữ model4, loại mechanic không hỗ trợ, rồi chuyển skill tương thích qua ThicMobKai.

ItemsAdder / custom model

Card, mob egg, rương boss, thẻ dungeon đều có thể đổi model và icon nếu admin muốn trang trí lại.

Trang 5

Cài đặt nhanh & FAQ

Một trang gọn cho lúc cần làm nhanh trên host, hoặc khi bạn muốn gửi cho người chơi / người mua plugin mà không cần giải thích dài dòng.

Quy trình cài đặt

  1. Thả jar vào thư mục plugins.
  2. Chạy server một lần để plugin tự sinh file YAML.
  3. Chỉnh `config.yml` và `message.yml` theo server của bạn.
  4. Set dungeon, set spawner, set boss, set card reward.
  5. Dùng GUI admin để tinh chỉnh cuối cùng.

Điểm nên bật

boss-damage.enabled world-boss.chest.enabled use-placeholderapi use-itemsadder-glyph spawner-hologram.enabled skill-tree.enabled 
# Gợi ý
world-boss:
  chest:
    enabled: true
    claim-limit: 5

FAQ

Có cần MythicMobs không?

Không bắt buộc. Plugin hướng tới việc giữ pack ModelEngine 4 nhưng chuyển skill tương thích qua ThicMobKai.

Có đổi lore / tên GUI mà không build lại được không?

Có. Dùng `message.yml` để đổi toàn bộ phần chữ hiển thị chính.

Boss thế giới có rơi rương không?

Có. Top 1-5 mở rương riêng theo rank, pool có thể chỉnh trong GUI admin.
